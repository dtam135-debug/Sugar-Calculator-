КАЛЬКУЛЯТОР СОВОКУПНОЙ САХАРНОЙ ДОЗЫ 
Инструкция для пользователей

ЧТО НУЖНО ДЛЯ РАБОТЫ
Приложение открывается как обычный файл в браузере.
Лучше всего использовать:
- Safari
- Google Chrome
- Яндекс Браузер

Нажмите «Добавить на экран "Домой"».

Результат: На телефоне появится иконка, которая открывает калькулятор как настоящее приложение.

ВАЖНО ЗНАТЬ
1. Приложение работает локально, интернет для расчетов не нужен.
2. История за 14 дней хранится в браузере на конкретном устройстве.
3. Если открыть калькулятор на другом телефоне или компьютере, история автоматически не перенесется.
4. Кнопка "Очистить все" в календаре удаляет всю историю за 14 дней.
5. Кнопка "Очистить все" под списком продуктов очищает только выбранный день.
